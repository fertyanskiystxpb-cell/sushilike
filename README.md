# Sushi Like — бот доставки суши для ВКонтакте

VK-бот для приёма заказов доставки суши. LongPoll, FSM-оформление заказа, админ-панель.

## Возможности

- Оформление заказа: блюда, время, приборы, доп. набор, адрес, телефон, оплата
- Предзаказ на завтра (вне часов работы)
- Оплата при получении или предоплата переводом
- Админ: приём/отмена заказов, шаблонные ответы, подтверждение оплаты
- Управление приёмом заказов и акциями

## Требования

- Python 3.10+
- Токен сообщества ВКонтакте (Long Poll API)

## Установка

```bash
# Клонировать / скопировать проект
cd "бот суши лайк"

# Создать виртуальное окружение (рекомендуется)
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate   # Linux/macOS

# Установить зависимости
pip install -r requirements.txt

# Настроить окружение
copy .env.example .env   # Windows
# cp .env.example .env   # Linux/macOS
# Заполните .env своими данными
```

## Запуск

```bash
python main.py
```

## Конфигурация (.env)

| Переменная | Обязательно | Описание |
|------------|-------------|----------|
| `VK_GROUP_TOKEN` | да | Токен сообщества (Long Poll) |
| `VK_GROUP_ID` | да | ID сообщества (число) |
| `VK_ADMIN_ID` | да | ID администратора |
| `PAYMENT_BANK` | да | Банк для перевода |
| `PAYMENT_ACCOUNT_NUMBER` | да | Номер карты/счёта |
| `PAYMENT_RECEIVER_NAME` | да | Получатель перевода |
| `PROMOS_FILE` | нет | Файл акций (по умолчанию promos.txt) |
| `GROUP_WALL_URL` | нет | Ссылка для кнопки «Акции» |
| `ORDER_ADDRESS_TEXT` | нет | Адрес заведения |
| `DELIVERY_ZONE_HINT` | нет | Подсказка по зоне доставки |
| `TIMEZONE_OFFSET_HOURS` | нет | Часовой пояс (по умолчанию 5 = UTC+5) |
| `DB_ENABLED` | нет | Сохранение в SQLite (по умолчанию False) |
| `DB_PATH` | нет | Путь к SQLite (по умолчанию sushi_like.db) |

Поддерживаются алиасы: `GROUP_TOKEN`, `GROUP_ID`, `ADMIN_ID`, `AYMENT_BANK`.

## Команды администратора

| Команда | Описание |
|---------|----------|
| `!прием выключить [причина]` | Отключить приём заказов |
| `!прием включить` | Включить приём заказов |
| `!акция добавить <текст>` | Добавить строку в акции |
| `!акция удалить <номер>` | Удалить строку акции по номеру |
| «📋 Текущие заказы» | Список активных заказов |
| «📊 Статистика за день» | Принято/отказано/сумма оплаченных за сегодня |

## Часы работы

- Пн–Пт: 10:00–21:00 (UTC+5 по умолчанию)
- Сб–Вс: 12:00–21:00

Вне часов работы предлагается предзаказ на завтра.
